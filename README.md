🚀 AutoML Interactive Trainer

A powerful and user-friendly Machine Learning Web App built using Streamlit that allows users to upload datasets, train models, evaluate performance, and make predictions — all without writing code.

📌 Overview

This project simplifies the complete Machine Learning pipeline by providing an interactive interface where users can:

Upload their own dataset (CSV)
Select input and output features
Choose ML models (Regression / Classification)
Tune parameters dynamically
Visualize results
Make real-time predictions
