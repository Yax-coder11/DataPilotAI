## 📌 Overview

ModelVerse AI is an **interactive AutoML web application** that allows users to:

✔ Upload datasets  
✔ Train ML models  
✔ Visualize performance  
✔ Make predictions  

All without writing code.

---

## ⚡ Features

<details>
<summary>📊 Data Handling</summary>

- Upload CSV dataset  
- Preview data instantly  
- Select X (features) and Y (target)  

</details>

---

<details>
<summary>🤖 Machine Learning Models</summary>

### 🔹 Regression
- Linear Regression  
- Multiple Regression  
- Polynomial Regression  

### 🔹 Classification
- KNN  
- SVM  
- Decision Tree  
- Random Forest  

</details>

---

<details>
<summary>⚙️ Customization</summary>

- Adjustable test size  
- Custom random state  
- Model-specific tuning:
  - K value (KNN)
  - Kernel (SVM)
  - Depth (Decision Tree)
  - Trees (Random Forest)

</details>
